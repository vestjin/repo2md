# REPO2MD - 项目转 Markdown

一个轻量级的移动端网页工具，可将项目压缩包（ZIP/TAR/GZ）解压后生成结构化的 Markdown 文档，方便在手机上查看、分享或归档项目代码。

## 📸 预览


## ✨ 功能特点

- **支持多种压缩格式**：ZIP、TAR、GZ、TGZ 及 `tar.gz`
- **文件树展示**：解压后以树形结构列出所有文件，可勾选需要导出的文件
- **后缀名筛选**：按文件后缀快速过滤显示的文件节点
- **生成 Markdown**：
  - 项目概览（项目名）
  - 目录结构树（仅包含勾选的文件）
  - 所有勾选文件的完整内容（自动跳过二进制文件）
- **安全检测**：自动识别常见二进制文件并跳过；敏感文件（如 `.env`、密钥等）导出前提醒
- **便捷操作**：支持一键复制 Markdown 到剪贴板、导出为 `.md` 文件、通过 Web Share API 分享
- **移动端优化**：触摸区域放大、深色模式适配、响应式布局

## 🚀 快速开始

### 在线使用
直接访问部署好的页面（例如 GitHub Pages），或下载项目到本地打开 `index.html` 即可。

### 本地运行
1. 克隆或下载本项目
2. 在浏览器中打开 `index.html`（推荐使用现代移动浏览器或 Chrome 开发者工具模拟移动设备）

### 使用步骤
1. 点击 **上传压缩包** 按钮，选择要转换的项目压缩包（ZIP/TAR/GZ 等）
2. 等待解压完成，文件列表将自动显示
3. 在文件树中勾选需要导出的文件（可展开文件夹）
4. 可选：使用 **后缀筛选** 快速过滤文件类型
5. 点击 **生成 Markdown**，等待处理完成
6. 在输出区域查看生成的 Markdown，可通过 **复制**、**导出** 或 **分享** 按钮保存

## 📦 支持的压缩格式

| 格式 | 说明 |
|------|------|
| ZIP | 标准 ZIP 格式 |
| TAR | 未压缩的 tar 包 |
| GZ / TGZ | gzip 压缩的 tar 包（包括 `.tar.gz`） |

> 注意：暂不支持 RAR、7z 等格式。

## 🛠 技术栈

- **HTML5 / CSS3**：结构样式，Flexbox/Grid 布局
- **JavaScript (ES6+)**：核心逻辑
- **第三方库**：
  - [jQuery](https://jquery.com/) + [jsTree](https://www.jstree.com/)：文件树渲染与交互
  - [JSZip](https://stuk.github.io/jszip/)：ZIP 解压
  - [pako](https://github.com/nodeca/pako)：GZIP 解压（用于 TAR.GZ）
- **CSS 特性**：深色模式适配、移动端触摸优化

## 📱 浏览器兼容性

- 现代移动浏览器（iOS Safari、Chrome for Android、Edge 等）
- 桌面浏览器亦可使用（但为移动端设计）
- 需要支持 `FileReader`、`Blob`、`Promise`、`async/await` 等 API

## ⚠️ 注意事项

- **文件数量限制**：一次性构建整个文件树，对于包含数万文件的超大压缩包可能导致页面卡顿，建议控制文件数量或分包处理。
- **二进制文件**：根据扩展名和文件头部自动检测二进制文件，生成 Markdown 时会跳过并添加 `[二进制文件，已跳过]` 提示。
- **敏感文件提醒**：如果勾选的文件路径包含常见敏感关键词（如 `.env`、`password` 等），生成前会弹出确认框。
- **隐私安全**：所有处理均在浏览器本地完成，不会上传任何文件到服务器。
- **深色模式**：跟随系统设置自动切换。
